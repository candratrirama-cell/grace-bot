# 🌸 Grace Bot WhatsApp

**Grace Bot** adalah bot WhatsApp gratis tanpa limit dengan berbagai fitur AI, group management, dan utility.

## ✨ Fitur
- 🤖 AI & Maker (WormGPT, Bratt, Blur Face, Waifu, Quote Anime, Screenshot)
- 👑 Group Management (Anti-link, Warn, Kick, Add, Promote, Demote, Tagall)
- 🔧 Utility (Sticker, Ping, Info, Time, Profile)

## 🚀 Deploy ke Render
1. Fork repository ini
2. Buka [render.com](https://render.com)
3. Connect ke GitHub
4. Deploy otomatis

## 📱 Cara Penggunaan
- Gunakan prefix `.` (titik)
- Ketik `.menu` untuk melihat semua fitur
